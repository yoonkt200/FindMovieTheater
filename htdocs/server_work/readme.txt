- 현재 폴더(server_work)에 들어있는 내용은, 
  본 프로젝트가 웹 앱으로써 동작하는 디렉토리 폴더 밖에서 데이터를 처리하거나 scraping 하는 background 파일들임.

- R script 파일들은 web상에서 scraping한 데이터를 json 데이터로 write 해주는 파일들임.

- 현재 폴더에 있는 파일들은 서버 컴퓨터에서 스케줄러를 이용해 실시간으로 실행해야 함.

- Database.csv는 서버 컴퓨터 Database에 있는 내용을 파일로 추출한 것. 서버를 옮길 시 이 데이터만 SQL로 업로드하면 됨.
